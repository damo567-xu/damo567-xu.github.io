📘 Markdown 学习与使用手册

本文档是 Markdown 语法学习资料与使用指南，内容涵盖基础语法、扩展语法、写作技巧及实用案例。
可直接导入任何 Markdown 编辑器或文档系统使用。

🧭 一、Markdown 是什么？

Markdown 是一种轻量级标记语言，用纯文本编写，通过简单符号实现：

  • 标题、列表、表格、链接、图片、代码块等格式；
  • 可在 GitHub、Notion、Typora、Obsidian、VSCode 等中直接使用；
  • 支持导出 PDF、HTML、Word 等多种格式。

🧩 四、Markdown 写作技巧

  • 使用空行区分段落；
  • 用三级标题分层内容；
  • 插入代码块讲解技术；
  • 使用表格对比信息；
  • 可嵌入 HTML 元素（如 <br>、<img>）；
  • 可结合 LaTeX 数学公式（如 $E=mc^2$）。
